# Generated Invitation Assets

The following decorative assets were generated for this project and do not depict real people or real venues:

- `paper-texture.webp`
- `outer-paper-texture.webp`
- `veil-texture.webp`
- `hero-botanical.webp`
- `floral-top-left.webp`
- `floral-bottom-right.webp`

The deterministic invitation shapes, monogram, seal, and envelope folds are implemented in code rather than generated as bitmap assets.

The following deterministic SVG ornaments are project-authored decorative assets:

- `ceremony-ornament.svg`
- `memory-envelope.svg`
